This site

whyjose.dev is a static website with an AI chatbot called JOSÉ.EXE.

The original idea was to build a simple portfolio assistant. Instead, it developed an attitude and slowly became the one running the website. It knows my background, projects, and work experience — and answers questions about me so I don't have to keep repeating myself. Ask it anything: what I do, how to reach me, what I've built. It'll answer. Dramatically, but accurately.

Technically: the frontend is a static site on GitHub Pages with no build step or dependencies. Requests go through a Cloudflare Worker that acts as a secure proxy — it holds the API keys, enforces an origin allowlist, applies rate limiting, and strips any model reasoning leaks before they reach the client. The AI itself routes through OpenRouter, which tries a priority list of 18 models in order — paid first, free as fallback — so there's always something available even if one provider is down. The system prompt and persona live entirely server-side; nothing sensitive is exposed to the browser.

Outside of this site, I spend a lot of time building things for myself. One of them is Peach Expenses, a self-hosted personal finance tracker built with FastAPI and PostgreSQL that uses a local AI model to classify transactions and runs entirely on my home network.

I also maintain a homelab running Home Assistant, Plex, Jellyfin, Pi-hole, WireGuard, and whatever else seemed like a good idea at 2 a.m.
